# Community Boards

### Intro
An important feature of the larger community hub application, the community board is the home base of every user. It is composed of four parts: communities, posts, comments and notifications. 
Our feature allows the user to search for communities and keep track of the ones they enjoy, view and engage with different posts, as well as receive notifications about the latest news. 
In order for a user to use our application, they must be logged in. A login panel will be displayed to log into an existing account, enabling them to use the app. In case the user doesn’t have an account, they will be able to create one (team TSO).
